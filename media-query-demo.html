Démo Media Queries — redimensionne la fenêtre !
Comment ça marche :
  • Desktop (> 768px) : sidebar à gauche, fond bleu
  • Tablette (≤ 768px) : colonnes empilées, fond vert
  • Mobile (≤ 480px) : layout compact, fond orange
L’indicateur en bas à droite affiche le breakpoint actif.

Sidebar

Sur desktop, je suis à gauche. Sur tablette/mobile, je passe au-dessus du contenu principal.

Contenu principal

Les media queries appliquent des styles différents selon la largeur du viewport. C’est la base du web responsive.

@media (max-width: 768px) {
  .layout { flex-direction: column; }
  .sidebar { width: 100%; }
}
